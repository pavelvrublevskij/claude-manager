Settings

~/.claude/settings.json
Global settings for Claude Code. Controls permissions, auto-updates, hooks, MCP, sandbox, and more. Precedence: managed > CLI args > local project > shared project > user (this file). Changes auto-reload in Claude Code.

Global CLAUDE.md

~/.claude/CLAUDE.md
Personal instructions loaded into every Claude Code session across all projects. Use for your preferences, role context, and default behaviors. Project-level CLAUDE.md files override these.
